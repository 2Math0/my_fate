# My Fate
Todo List app

## Description
My Fate is an app of making your own fate by clicks, set a task and the app will make you do it (reminders)

## Features
This app will include : 
- Local Storage (Hive)
- Push Notifications
- State Management
- Mvvm Architecture
- Add,Edit,Delete,Mark as done Tasks
- categorized tasks
- tasks timeline
- reponsive design
- light/dark themes
